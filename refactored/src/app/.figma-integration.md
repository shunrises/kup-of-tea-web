# Figma MCP 서버 연동 가이드

이 프로젝트는 Figma MCP 서버를 통해 UI 디자인을 가져올 수 있도록 구성되어 있습니다.

## 사용 방법

### 1. Figma 디자인 파일 준비

- Figma에서 디자인 파일을 준비합니다
- 디자인 파일의 `fileKey`와 노드의 `nodeId`를 확인합니다

### 2. MCP 도구를 통한 디자인 가져오기

Figma MCP 서버의 다음 도구들을 사용할 수 있습니다:

- `get_design_context`: UI 코드 생성
- `get_metadata`: 노드 메타데이터 가져오기
- `get_screenshot`: 스크린샷 생성
- `get_code_connect_map`: Code Connect 매핑 가져오기

### 3. 예시

Figma URL이 주어진 경우:

```
https://figma.com/design/pqrs/ExampleFile?node-id=1-2
```

이 경우:
- `fileKey`: `pqrs`
- `nodeId`: `1:2` (URL에서는 `1-2`이지만 노드 ID는 `1:2` 형식)

### 4. 컴포넌트 생성

Figma에서 가져온 디자인을 기반으로 컴포넌트를 생성할 수 있습니다:

```typescript
// components/ExampleComponent.tsx
// Figma 디자인을 기반으로 생성된 컴포넌트
```

## 주의사항

- Figma MCP 서버가 올바르게 설정되어 있어야 합니다
- 디자인 파일에 접근 권한이 있어야 합니다
- 코드 생성 후 필요에 따라 수정이 필요할 수 있습니다
